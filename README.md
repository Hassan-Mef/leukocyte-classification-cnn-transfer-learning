# Leukocyte Classification using CNN and Transfer Learning

This project focuses on the classification of white blood cell (leukocyte) images using Convolutional Neural Networks (CNNs). The work is inspired by a research paper published in Scientific Reports (Nature).

The objective of this project is to explore how deep learning models can be used to automatically identify different types of white blood cells, which is an important step in medical diagnosis.

We will implement and compare multiple CNN architectures, including:
- AlexNet
- VGG16
- GoogLeNet (Inception)

These models will be applied using transfer learning to improve performance and reduce training time.

---

## Project Scope

- Image classification of leukocytes (WBCs)
- Multi-class classification problem
- Comparative analysis of CNN models
- Efficient training using a reduced dataset

---

## Dataset

The original dataset contains over 171,000 images across 21 classes.  
For this project, we will use a subset consisting of selected leukocyte classes with a limited number of images per class to ensure manageable computation and faster training.
